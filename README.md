# 🚀 AI Academy – Interactive AI Learning Platform

An intelligent, interactive platform that makes learning AI/ML simple, practical, and engaging using real-time AI assistance.

## 📌 Problem

Learning Artificial Intelligence and Machine Learning is often:
- Complex
- Theoretical
- Not interactive

Students struggle to apply concepts in real-world scenarios.

## 💡 Solution

AI Academy provides:
- Interactive learning modules
- Real-time AI tutor (Gemini API)
- Hands-on quizzes
- Practical explanations with examples

It transforms passive learning into an active experience.

## 🔥 Features

- 🔐 Modern Login Interface
- 📊 Interactive Dashboard
- 📘 AI/ML Learning Modules
- 🤖 AI Tutor (Gemini-powered)
- 🧠 Smart Quiz System with scoring
- ⚡ Real-time AI responses
- 🎨 Modern UI using Tailwind CSS

## 🤖 AI Integration

We used Google Gemini API to:

- Explain AI concepts in simple language
- Provide real-world examples
- Act as an intelligent tutor
- Enhance user interaction

Prompt Engineering:
- Structured explanations
- Step-by-step breakdown
- Beginner-friendly + practical examples

## 🛠️ Tech Stack

- React (Vite)
- Tailwind CSS
- Google Gemini API
- LocalStorage

## ⚙️ Run Locally

### Prerequisites:
- Node.js

### Steps:
1. Install dependencies:
   npm install

2. Add your Gemini API key:
   Create a `.env.local` file and add:
   GEMINI_API_KEY=your_api_key_here

3. Run the app:
   npm run dev
